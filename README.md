BankCard OOP Project

This project demonstrates Object-Oriented Programming (OOP) concepts using Java. It was developed as part of a coursework assignment to practice inheritance, class design, and OOP principles. The project was created and run using BlueJ.

Project Overview

Superclass: BankCard – Represents a general bank card.

Subclasses:

DebitCard – Represents a debit card with specific behaviors.

CreditCard – Represents a credit card with specific behaviors.

Demonstrates inheritance, class hierarchy, and OOP design in Java.

Features

Creation of BankCard objects and their subclasses.

Implementation of specific methods for DebitCard and CreditCard.

Illustrates use of constructors, methods, and polymorphism in Java.

Technologies Used

Java (Object-Oriented Programming)

BlueJ IDE

Usage

Open the project in BlueJ.

Compile the classes (BankCard, DebitCard, CreditCard).

Run the project and test the behaviors of DebitCard and CreditCard objects.

Learning Outcomes

Understand OOP concepts like inheritance and polymorphism.

Gain hands-on experience with Java class design.

Familiarity with BlueJ IDE for Java development.
